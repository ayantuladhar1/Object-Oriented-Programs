*******************************************************
*  Description of the program
*******************************************************
This program creates a fantasy role-playing game where its has four differnt types of creatures: humans, cyberdemons, balrogs, and elves.
The game is played in one round combat between all four creatues depending and displays how much damages they can inflict and sustain.


*******************************************************
*  Source files
*******************************************************

Name:  main.cpp
   Main program. This is the main program which contains a driver to test the classes for all four creatures and their roles. It also creates 
an object for each type of creature and repeatedly ouputs the results.

Name:  Header.h
   Contains the definition of Header for Creature, Demon, Cyberdemon, Human, Elf and Balrog class.  

Name:  gridclass.cpp
   Contains the definition of Header and Constructur for Creature, Demon, Cyberdemon, Human, Elf and Balrog class.
