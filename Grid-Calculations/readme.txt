*******************************************************
*  Description of the program
*******************************************************
This program helps to build two grids where the width and length are denoted by X and Y respectively where the user
inputs the value in X and Y to determine the size of the grids. For example 2 by 2 grid, 3 by 3 grid etc.
*******************************************************
*  Source files
*******************************************************

Name:  main.cpp
   Main program. This is the main program which creates and fills each cell of both grids with a '0' and randomly fill 1/3rd of the
grid's cell with '1'. 

Name:  Header.h
   Contains the definition of Header for Grid class.  

Name:  gridclass.cpp
   Contains the definition of Header and Constructur for Grid class. It also contains filling function for each cell.
